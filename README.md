# 🏟 Smart Stadium AI Assistant

> Navigate smarter. Skip crowds.

Built as part of the **PromptWars challenge**, this project focuses on improving the physical event experience in large stadiums using AI-driven decision making.

---

## 🚀 Problem Statement

Large-scale sporting events often face challenges such as:

* Overcrowded entry/exit points
* Long waiting times at food stalls
* Lack of real-time guidance

These issues reduce overall attendee experience and can create safety concerns.

---

## 💡 Solution

Smart Stadium AI Assistant is a web-based system that helps users:

* Find the least crowded routes inside a stadium
* Check live crowd conditions
* Get instant answers to common queries
* Navigate safely during emergencies

---

## 🧠 Key Features

### 🔹 Smart Route Planner

* Suggests optimal path based on crowd density
* Dynamically avoids highly crowded gates
* Displays route, estimated time, and crowd level

### 🔹 AI Assistant

* Handles user queries such as:

  * Nearest washroom
  * Crowd status
  * Food availability
  * Exit guidance
* Provides contextual responses based on live data

### 🔹 Live Stadium Dashboard

* Shows crowd percentage for each gate
* Highlights overcrowded areas with alerts
* Updates automatically to simulate real-time conditions

### 🔹 Emergency Mode 🚨

* Detects emergency-related queries
* Guides users to the safest and least crowded exit

### 🔹 Google Maps Integration

* Embedded map for real-world stadium context

---

## ⚙️ Tech Stack

* HTML
* CSS
* JavaScript
* Google Maps Embed

---

## 🧩 How It Works

* Crowd data is simulated and updated periodically
* The system evaluates all gates and selects the least crowded path
* The assistant processes queries using keyword-based logic
* UI updates dynamically to reflect real-time conditions

---

## ⚠️ Assumptions

* Crowd data is simulated (no real sensors used)
* Stadium layout is simplified
* AI assistant uses rule-based logic

---
